Comparison between class and id in HTML/CSS

Feature class id
Purpose Applies styles to multiple elements Applies styles to one unique element
Reusability Can be reused on many elements Should be used only once per page
Selector in CSS Uses dot: .className Uses hash: #idName
Syntax in HTML <div class="box"> <div id="header">
Best Use Case Group styling for repeated elements Target unique sections like headers/footers
JavaScript Access getElementsByClassName("box") getElementById("header")
Priority in CSS Lower specificity Higher specificity
CSS Example .box { color: red; } #header { color: blue; }
